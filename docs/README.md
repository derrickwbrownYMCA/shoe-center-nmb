# Shoe Center NMB Project Docs

This folder is the working source of truth for the project.

Use these documents in this order:

1. `website-roadmap.md`
   The phased plan to turn the Shopify theme into a high-converting, market-leading storefront.
2. `catalog-migration-blueprint.md`
   The staged plan for importing and validating a 10,000-SKU catalog.
3. `product-data-architecture.md`
   The Shopify data model for products, variants, metafields, collections, and filters.

Guiding principles:

- Keep `main` stable.
- Make changes on feature branches.
- Treat catalog migration as a data project, not just a CSV upload.
- Design the storefront and product data model together so merchandising, search, and filters scale cleanly.
