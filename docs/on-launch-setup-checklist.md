# On Launch Setup Checklist

This file pairs the catalog import work with the remaining launch tasks so each product family can move cleanly from draft to published.

## Working File

- `On Product Setup Tracker.csv`

## Recommended Workflow

1. Start with Wave 1 product families only.
2. Assign hero and gallery images by handle.
3. Fill in body copy and size notes.
4. Confirm metafields for support, cushioning, waterproof, and width guidance.
5. Mark `ready_for_import` only after images and copy are paired.
6. Mark `ready_for_publish` only after the Shopify product page is reviewed on desktop and mobile.

## Launch Advice

- Use `On Inventory - shopify-wave1-core.csv` for the first import pass.
- Treat the tracker as the source of truth for which handles are fully dressed with copy and images.
- Expand from core to full Wave 1 only after the first import looks right in the live theme.
