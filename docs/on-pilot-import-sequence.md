# On Pilot Import Sequence

Recommended order for the first Shopify test imports based on flagship value, launch risk, and storefront usefulness.

## Pilot Batch 1

- `on-cloud-6-women`
- `on-cloud-6-men`
- `on-cloudrunner-2-women`
- `on-cloudrunner-2-men`

Reason: these represent your highest-value On story and will expose the biggest variant, width, and merchandising issues first.

## Pilot Batch 2

- `on-cloudswift-4-women`
- `on-cloudswift-4-men`
- `on-cloud-6-women-waterproof`
- `on-cloud-6-men-waterproof`

Reason: these add lower-risk product families after the flagship structure is validated.
